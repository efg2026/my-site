# East Financial Group Website

This is a single-page responsive website built with HTML and Tailwind CSS.

## Getting Started

1. Open `index.html` in your web browser to view the site.
2. The website uses Tailwind CSS via CDN, so an internet connection is required to load styles and icons.
3. Images are sourced from Unsplash and Google Drive.

## File Structure

- `index.html`: The main website file containing HTML, CSS configuration, and JavaScript.

## Editing

To edit the site, simply modify the `index.html` file in any code editor (like VS Code, Sublime Text, or Notepad).
